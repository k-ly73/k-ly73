PhotoBum, a miniature social media application

PhotoBum

PhotoBum is a mini social media, instagram-like application. Implemented features include posting images, commenting on posts, and like button functionality, and google sign-in for users to participate. This application was developed using ReactJS framework with Google Firebase backend storage.

BobaFrenzy

This little android app is made as a creative simulation for people to selectively order milk tea drinks as well as an interactive page to control widgets that allow users to "virtually create" their own drinks. This mobile application was developed using Flutter as the main framework, with the Dart Programming language as support and Google Firebase for backend storage.

TheShuffling, a web app for MTG players

TheShuffling

A web application that caters to users who play Magic the Gathering. This website allows users to catalog a collection of specific cards within their posession, making it easier to keep track of their deck without having to completely search out their deck. Other implemented features include global chatting system. This application was developed using standard web programming tools with NodeJS to read the API data for Magic the Gathering cards and MongoDB as database storage
